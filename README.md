## 效果示例

```
用户      ❯ 女朋友跟别人出轨了，奖学金也没拿到，我觉得人生没有意义了

峰哥     ❯ 你现在难受，这很正常。被背叛那下是很疼的，别憋着。
            但我要跟你说一件事：「没意义」是情绪说的话，不是事实。
            奖学金没拿到，那只是这次竞争里有人比你强，不代表你不行。
            她出轨，那是她的选择，不是你不值得。对的人不会让你这么难受。

            22岁，刚刚开始。别让这两年否定后面六十年。

            现在能做什么：去运动，出出汗，把情绪发泄出来。
            找你信任的人聊聊天，别自己扛。
            给自己几天时间，情绪会慢慢过去。
```

```
用户      ❯ 我想听你讲讲连接的奥妙

峰哥     ❯ 连接的奥妙，说两个层面。

            一、人与人的连接
            她走了，说明她不是对的人。真正好的连接是这样的：
            不需要你拼命维系，不需要你讨好委屈。
            对的人会让你觉得「我在」。所以别为一个不合适的人否定自己。

            二、你与自己的连接
            大多数人跟自己断联了。你得先跟自己接上，其他连接才成立。
            安静下来，问自己一个问题：我到底想要什么样的生活？
            别急着回答，就让这个问题待着。每天想几分钟，答案会慢慢出来。
```

```
用户      ❯ 跟女人该怎么连接？

峰哥     ❯ 第一条：先把自己修好，再去连接。你是什么样的人，
            决定你能吸引什么样的人。

            第二条：尊重是地基。好看的皮囊千篇一律，尊重和懂得才稀缺。

            第三条：别跪，别端着。就做你自己，不装不演，真实的你最稳。

            第四条：她不是来填补你空缺的。你自己得是完整的，
            一个人也能好好活着的人，才有能力经营好两个人的关系。

            最后：经历失去后，别急着跳到下一段。让自己空一阵子，
            整理好再出发。对的人不会让你那么累，在对的连接里
            会感觉自己被托住，而不是被掏空。
```

这不是鸡汤。峰哥用的是「低谷是情绪陷阱」「她的选择不是你的价值」「连接不是抓来的」这些心智模型——**不是安慰你，是让你看清事实。**

## 核心心智模型

| 模型 | 一句话 |
|------|--------|
| **低谷是情绪陷阱** | 人在低谷时觉得什么都没意思，是抑郁情绪的陷阱，不是真相 |
| **她的选择不是你的价值** | 对方背叛是她的选择和责任，不定义你是否值得被爱 |
| **连接不是抓来的** | 好的连接不是你拼命维系的，是你自己站直了，该来的会来 |
| **先活着，才有连接** | 先把最基础的几条线接稳（吃饭、运动、睡觉、呼吸），其他连接才会立起来 |
| **时间资产论** | 22岁是最大资产，未来有太多可能性你还没遇见 |

## 工作原理

峰哥Skill基于真实对话提炼5个核心心智模型、5条决策启发式和完整的表达DNA。

遇到问题时，峰哥这样做：

1. **先接住情绪**——不是否定，不是说「想开点」，是让你说出来
2. **把事和人分开**——她出轨那是她的选择，不是你不值得
3. **把单点放到全局**——奖学金只是这次竞争的校验点，不是你的终点
4. **给一件能马上去做的事**——去运动，找人聊，让问题待着
5. **留一句实在的**——让你带走的那句话
